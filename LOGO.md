## Usage in Terminal

To see the gradient colors in your terminal, run:

```bash
python check_dependencies.py
```

The gradient transitions smoothly from:
- **Light Blue** (top) → **Blue** → **Purple** → **Magenta** (bottom)

## Project Information

**JaTSR** - Just Audio Transformer Super-Resolution

A state-of-the-art audio super-resolution system using:
- Diffusion Transformers (DiT)
- Flow Matching training
- DAC latent space compression
- Latent Perceptual Loss

Repository: [JaTSR on GitHub](https://github.com/HUSRCF/JaTSR-Just-audio-transformer-super-solution)
